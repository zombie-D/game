Jeu de Deviner le Nombre

L'objectif du jeu est très simple : sur la première case, vous entrez votre nom. Sur les 3 autres cases en bas, vous choisissez 3 chiffres différents. Si au moins 1 des 3 chiffres que vous avez entrés est le chiffre gagnant, alors vous aurez un message qui vous dira que vous avez gagné un ticket. Dans le cas où vous ne devinez pas le bon chiffre, alors vous aurez un message qui vous demandera de réessayer.
Le nombre à deviner est situé entre 0 et 30. Lorsque vous réussissez, faites une capture d'écran du message de réussite et envoyez-le dans le grand groupe d'ESCAE.


Bonne chance à tous, et que la chance soit de votre côté! Yosh xD
